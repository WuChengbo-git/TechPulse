# TechPulse Release Notes

## Version 1.0.2 - Major Feature Update
**发布日期**: 2025-08-20

### 🎯 核心功能概述
TechPulse 是一个技术情报收集和分析平台，帮助开发者实时掌握最新的技术动态和趋势。

### 🚀 新增功能

#### 1. 智能数据抓取系统
- **多数据源支持**: 集成 GitHub、arXiv、Hugging Face、Zenn 四大技术平台
- **时效性过滤**: 自动抓取最近30天内活跃的项目和研究
- **智能关键词匹配**: 基于AI、机器学习、深度学习等热门技术关键词进行精准筛选
- **增量更新机制**: 支持全量和增量数据收集，避免重复抓取

#### 2. AI 驱动的内容分析
- **Azure OpenAI 集成**: 使用先进的 GPT 模型进行内容分析
- **8维度项目评估**: 
  - 技术创新性评分
  - 实用性和可用性分析
  - 学习价值评估
  - 技术栈分类
  - 中文标签生成
  - AI应用领域识别
  - 推荐理由生成
  - 适用人群分析
- **智能摘要生成**: 自动生成项目的中文概要和技术要点

#### 3. 自动化任务调度
- **后台定时任务**: 每6小时自动执行数据收集
- **灵活配置**: 支持通过环境变量调整收集间隔
- **任务状态监控**: 实时跟踪数据收集进度和状态

#### 4. 智能聊天助手 🤖
- **网页内容分析**: 输入任意网页链接，AI自动提取和分析内容
- **多轮对话**: 基于分析结果进行深度问答交流
- **内容类型识别**: 自动识别技术博客、项目文档、研究论文等
- **实时分析**: 支持URL验证和内容实时抓取

#### 5. 数据可视化分析 📊
- **多维度图表**: 柱状图、折线图、饼图、词云图等多种可视化形式
- **时间范围筛选**: 支持7天、14天、30天不同时间维度分析
- **技术趋势分析**: 实时展示热门技术标签和增长趋势
- **项目热度排行**: 基于Star数和活跃度的项目排名
- **数据源分布**: 可视化展示不同平台的数据贡献比例

### 🛠 技术架构升级

#### 后端技术栈
- **框架**: FastAPI + SQLAlchemy + Alembic
- **AI服务**: Azure OpenAI GPT-3.5/4.0
- **任务调度**: Schedule + 异步任务处理
- **数据抓取**: 
  - GitHub API v4 (GraphQL)
  - arXiv API + feedparser
  - Hugging Face API
  - Zenn RSS feeds
- **Web内容提取**: BeautifulSoup4 + requests
- **数据库**: SQLite (可扩展至PostgreSQL/MySQL)

#### 前端技术栈
- **框架**: React 18 + TypeScript + Vite
- **UI组件**: Ant Design 5.x
- **图表库**: @ant-design/charts + ECharts
- **路由**: React Router
- **状态管理**: React Hooks
- **时间处理**: dayjs
- **响应式设计**: 全平台适配

### 🌐 部署和运维

#### 系统要求
- **操作系统**: Linux/macOS/Windows
- **Python**: 3.8+
- **Node.js**: 16+
- **内存**: 最小2GB，推荐4GB+
- **存储**: 最小1GB可用空间

#### 快速部署
```bash
# 1. 克隆项目
git clone <repository-url>
cd TechPulse

# 2. 配置环境变量
cp backend/.env.example backend/.env
# 编辑 backend/.env 文件，填入API密钥

# 3. 一键启动
chmod +x start.sh
./start.sh
```

#### 服务访问
- **前端界面**: http://localhost:5174
- **后端API**: http://localhost:8001
- **API文档**: http://localhost:8001/docs
